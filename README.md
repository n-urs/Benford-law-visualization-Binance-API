# Benford-law-visualization-Binance-API
PyQt6 desktop tool to test Benford’s law (two leading digits 10–99) on Binance Spot &amp; USDT-Perp Futures. Fetches aggTrades or RAW historical trades, analyzes qty and USDT size, overlays Benford probabilities, shows χ², and live-updates every 10k rows. Start/Stop/Continue/End controls, custom control window, CSV written next to the script.
